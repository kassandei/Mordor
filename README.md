# Mordor

Text-based RPG game - University project

## Download

```bash
git clone https://github.com/kassandei/Mordor.git
cd Mordor
```

## Compile & Run

```bash
make
```

## Only compile

```bash
make compile
```

## Doxygen Documentation

The documentation is already generated in the `html/` folder. To view it:

```bash
open html/index.html
```

Or open the file `html/index.html` with your browser.

To regenerate the documentation:

```bash
doxygen
```
