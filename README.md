# Mordor

Text-based RPG game - University project

## Download

```bash
git clone https://github.com/kassandei/Mordor.git
cd Mordor
```

## Compile & Run

```bash
make
```

## Only compile

```bash
make compile
```

## Doxygen Documentation

To generate the documentation:

```bash
doxygen Doxyfile
```

Then open `html/index.html` 

```bash
open html/index.html
```
